## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Custom properties, Flexbox/Grid, Keyframe animations)
- **Design System**: High-fidelity recreation based on modern web design principles.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Chetanvaidada/Olive_mimic.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Olive_mimic
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 📸 Preview

The application features a sophisticated auto-sliding carousel where products like *Dave's Killer Bread*, *San Pellegrino*, and *Larabar* are showcased with their unique nutritional scores and "Oliver Says" insights.
